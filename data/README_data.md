# Included Processed Data

This repository includes selected processed files that are central to the saved modeling stages.

Included:
- aggregated compound-level interaction table
- early high-confidence inhibitor dataset
- inhibitor/substrate v2 datasets
- final 522-compound inhibitor dataset with SMILES
- saved Phase0 benchmark outputs
- representative XAI figures

Excluded on purpose:
- large intermediate chunk files
- raw batch outputs
- empty or incomplete result folders
- most binary graph dataset artifacts
