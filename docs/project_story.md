# Project Story

This project began as a literature-driven attempt to organize P-gp-related compound interactions into a structured modeling dataset.

The notebook history reflects how the work evolved over time:
1. `SohnProject1`: original literature-mining workflow and early inhibitor-focused modeling
2. `NewPgp`: later expansion into inhibitor/substrate-oriented problem framing
3. `8_15_All_NewPgp`: more polished inhibitor-focused reorganization using a final 522-compound dataset and cleaner benchmark reporting

A smaller auxiliary ChEMBL notebook exists in the original archive, but it is intentionally excluded from the main portfolio package because the current saved outputs are centered on the literature-derived pipeline.
